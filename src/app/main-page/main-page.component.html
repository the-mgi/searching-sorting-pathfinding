What is an Algorithm?

As computers does not possess intelligence on their own therefore it is required for us humans to provide instructions in such way which a computer can understand easily which are given in any given Programming Language. Therefore, An Algorithm can be defined as the set of instructions that we pass on to th computer to do any task in its COMPUTING LIMITS It ranges from simply finding out the LCM of numbers to complex Deep Learning Algorithms such as Logistic Regression.

There are tons of algorithms for Sorting, Searching and Pathfinding.

The Whats , Whys and Algorithms that fall into these categories are given below.

Sorting Algorithms

What?

Sorting Algorithms is specifically used to arrange the un-organized elements in a given collection. This arrangement can be based on ASCII Codes or simply which number is greater than the other on a number line.

Why?

It proves to be useful in different areas such as for arranging the Book Catalogue in an online E-Book Store or Information of People arranged in alphabetical order which is useful for faster retrieval of data from any of these Information Stores.

Searching Algorithms

What?

Searching Algorithms are used to find an element in any given Data Structure, of which there are different algorithms for each. They are used to retrieve an element or to find their index in the Data Structure. For their purposes they are divided into two major categories:

  • Sequential Search

    In Sequential Search category, the array is completely traversed and each element is sequentially compared with the KEY element which is to be found. Also the array could be in SORTED or UNSORTED order.Example: Linear Search

  • Interval Search

    In Interval Search category, the algorithms are all designed in such a way that they require the arrays to be SORTED to give out the accurate results. These algorithms does not sequentially traverse the array and does have LOW RUNNING TIME (which we will discussed later) than that of Sequential Search ones, giving them advantage. Example: Binary Search

Why?

Search Algorithms are quite a great deal in computer science whose uses then branches to different disciplines. For instance, one of the world's biggest Tech Giant Google is all about providing the accurate results according to one's query. And one of the biggest challenge is to find any given element in reasonable amount of time. Moreover not every algorithm is efficient for each use case.
